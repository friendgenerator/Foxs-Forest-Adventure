# Fox's Forest Adventure

Simple level of a forest themed platformer



Controls:

A or D (Left / Right Arrow)   Horizontal movement

W (Up Arrow)                  Jump

Space Bar                     Attack

R                             Restart (when prompted)



With assets from:

https://brullov.itch.io/oak-woods

https://lyaseek.itch.io/miniffanimals

https://elthen.itch.io/2d-pixel-art-leaf-elemental-sprites
